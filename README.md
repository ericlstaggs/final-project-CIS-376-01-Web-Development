# Muscle Shoals Sailing Club – Website Redesign

A modern, responsive overhaul of the [Muscle Shoals Sailing Club](https://www.muscleshoalssailingclub.org/) digital presence. This project migrates the existing legacy architecture to a high-performance **React.js** application styled with **Bootstrap 5**.

## 🎯 Project Objectives
* **Modernize UI/UX:** Replace the dated interface with a clean, nautical-themed aesthetic.
* **Mobile-First Design:** Ensure full responsiveness for members accessing the site from the docks or on the water.
* **Performance:** Improve page load speeds and navigation fluidly via React’s component-based architecture.
* **Accessibility:** Enhance readability and navigation for a diverse club membership.

---

## 🛠 Tech Stack

| Technology | Purpose |
| :--- | :--- |
| **React.js** | Frontend library for UI components and state management. |
| **Bootstrap 5** | Layout framework for responsive grids and pre-built UI elements. |
| **SASS/CSS3** | Custom styling to align with the club’s branding. |
| **React Router** | Client-side routing for seamless page transitions. |

---

## 🚀 Key Features

* **Dynamic Event Calendar:** A user-friendly view of upcoming regattas, club meetings, and social events.
* **Responsive Gallery:** Showcasing club life and sailing events using Bootstrap carousels and lightboxes.
* **Membership Portal:** Streamlined information and forms for prospective and current members.
* **Weather Integration:** Real-time wind and weather conditions for the Muscle Shoals area.

## 💻 Getting Started

### Prerequisites
* **Node.js** (v16.0 or higher)
* **npm** or **yarn**

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/mssc-redesign.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd mssc-redesign
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start the development server:**
   ```bash
   npm start
   ```
   The site will be viewable at `http://localhost:3000`.
